## File Description
This Python program implements a Bank Account Management System that allows users to perform basic banking operations such as creating an account, depositing money, withdrawing money, and checking account balance using Object-Oriented Programming (OOP) concepts.

## Features
- Create a new bank account
- Deposit money
- Withdraw money (with balance check)
- View account details
- Check current balance
- Menu-driven program

 ## Concepts Used
- Python OOP (Class & Object)
- Constructors (_init_)
- Instance variables
- Conditional statements
- Loops
- User input handling
  
## How the Program Works
- User selects an option from the menu
- The program performs the selected banking operation
- Data is stored temporarily during execution
- Program continues until the user exits
